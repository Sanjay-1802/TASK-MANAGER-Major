# TASK-MANAGER-Major

# Project Overview
This project is a Task Manager web application developed as a major project. The main purpose of this project is to help users organize and manage their daily tasks in a simple and user-friendly way. Users can create, update, delete and keep track of their tasks through a clean interface.
The project is developed using HTML, CSS, JavaScript, Node.js, Express.js and MongoDB. The frontend provides the user interface while the backend handles all server-side operations and database management.

# Features
* User Registration and Login
* Secure Authentication
* Create New Tasks
* Update Existing Tasks
* Delete Tasks
* Mark Tasks as Completed
* Responsive User Interface
* Data Stored in MongoDB Database

# How to Run the Project
1. Download or clone this repository.
2. Open the backend folder.
3. Install the required packages.
4. Create a .env file and add the required environment variables.
5. Start the server.
6. Open the frontend in your browser or run the project using the provided live deployment.
